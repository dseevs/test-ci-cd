## Offline Nunito fonts

Nunito font files for **offline** use. Loaded by `@font-face` rules in `src/app/globals.css`.

These files come from [google-webfonts-helper](https://gwfh.mranftl.com/fonts/nunito) (latin subset, woff2):

- `nunito-v32-latin-200.woff2` / `nunito-v32-latin-200italic.woff2`
- `nunito-v32-latin-300.woff2` / `nunito-v32-latin-300italic.woff2`
- `nunito-v32-latin-regular.woff2` (= weight 400) / `nunito-v32-latin-italic.woff2`
- `nunito-v32-latin-500.woff2` / `nunito-v32-latin-500italic.woff2`
- `nunito-v32-latin-600.woff2` / `nunito-v32-latin-600italic.woff2`
- `nunito-v32-latin-700.woff2` / `nunito-v32-latin-700italic.woff2`
- `nunito-v32-latin-800.woff2` / `nunito-v32-latin-800italic.woff2`
- `nunito-v32-latin-900.woff2` / `nunito-v32-latin-900italic.woff2`

If you add or remove files, update the matching `@font-face` blocks in `src/app/globals.css`.
